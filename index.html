Configuration Setup

OpenManus requires configuration for LLM providers and optional services.

1

Choose Your LLM Provider

Select and configure your preferred language model provider:

ü§ñ Google Gemini

Recommended for general use

model = "gemini-2.5-pro"
üß† OpenAI GPT

Popular choice with excellent performance

model = "gpt-4o"
üé≠ Anthropic Claude

Great for reasoning and analysis

model = "claude-3-7-sonnet-latest"
2

Setup Configuration File

Copy the example configuration and add your API key:

cp config/config.example.toml config/config.toml
3

Install Dependencies

Install required Python packages:

pip install -r requirements.txt
4

Run OpenManus

Start the agent with one of these commands:

# Basic agent
python main.py

# Multi-agent flow
python run_flow.py

# MCP mode
python run_mcp.py

When running in Bolt's WebContainer environment:

  • Docker-based sandbox features are not available
  • Browser automation tools have limited functionality
  • Only standard library Python packages are supported
  • Some system-level operations may not work

For full functionality, deploy to a server environment with Docker support.
